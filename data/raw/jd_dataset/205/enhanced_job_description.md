## Job Title
Python Backend Developer

## Location
N/A

## Client Industry
Information Technology Services / Technology Solutions

## Detailed Responsibilities
- Design, implement, and maintain scalable backend services and APIs using Python and related frameworks.
- Collaborate closely with front-end developers, DevOps teams, and product managers to deliver integrated solutions.
- Write clean, efficient, and well-documented code adhering to best practices and coding standards.
- Optimize application performance, troubleshoot issues, and ensure high availability of backend systems.
- Participate in code reviews, contribute to architectural decisions, and mentor junior team members.
- Integrate third-party APIs and data sources as required by business needs.
- Ensure application security through rigorous testing and implementation of secure coding practices.
- Stay updated on emerging backend technologies and advocate for their adoption where relevant.

## Skill Requirements
- Bachelor’s degree in Computer Science, Information Technology, or a related field.
- Proven experience (3+ years) in backend development using Python.
- Strong knowledge of Python frameworks such as Django, Flask, or FastAPI.
- Proficiency in RESTful API design and development.
- Experience with relational and non-relational databases (e.g., PostgreSQL, MySQL, MongoDB).
- Familiarity with version control systems (preferably Git).
- Understanding of containerization technologies (e.g., Docker) and cloud platforms (AWS, Azure, or GCP).
- Excellent problem-solving skills and attention to detail.
- Strong communication skills and ability to work effectively in a multinational team environment.

## Other Requirements
- Experience with CI/CD pipelines and automated testing frameworks.
- Exposure to microservices architecture and event-driven systems.
- Knowledge of Agile methodologies and tools (e.g., Jira, Confluence).
- Certifications in Python or cloud technologies are a plus.
