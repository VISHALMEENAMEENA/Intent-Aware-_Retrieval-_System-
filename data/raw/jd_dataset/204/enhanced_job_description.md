## Job Title
Python Backend Developer

## Location
N/A

## Client Industry
Technology

## Detailed Responsibilities
- Design, develop, and maintain efficient, reusable, and reliable Python code for backend applications.
- Build and optimize RESTful APIs using FastAPI to enable seamless integration with front-end systems and third-party services.
- Collaborate with cross-functional teams, including front-end developers, product managers, and QA engineers, to deliver high-quality software solutions.
- Participate in code reviews, contribute to best practices, and ensure that the codebase is robust and maintainable.
- Troubleshoot, debug, and resolve technical issues across the development lifecycle.
- Write and maintain technical documentation to support ongoing development and knowledge sharing.
- Implement security and data protection measures in compliance with industry standards.
- Support the deployment of applications in cloud or on-premise environments as required.

## Skill Requirements
- Bachelor’s degree in Computer Science, Information Technology, or a related field.
- 2-4 years of hands-on experience in backend development with Python.
- Proficiency in FastAPI framework for building RESTful APIs.
- Strong understanding of software development principles, data structures, and design patterns.
- Experience with version control systems such as Git.
- Familiarity with relational and NoSQL databases.
- Good communication skills and ability to work collaboratively in a team environment.

## Other Requirements
- Experience with containerization technologies such as Docker or Kubernetes.
- Exposure to cloud platforms like AWS, Azure, or Google Cloud.
- Familiarity with CI/CD pipelines and DevOps practices.
- Knowledge of other Python web frameworks (e.g., Flask, Django) is a plus.
- Certification in Python or related technologies.
