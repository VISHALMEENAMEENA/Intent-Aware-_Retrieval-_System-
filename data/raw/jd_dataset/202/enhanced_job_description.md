## Job Title
Python Automation Engineer

## Location
N/A

## Client Industry
Technology

## Detailed Responsibilities
- Design, develop, and maintain scalable Python automation scripts for web application testing using Selenium and related tools.
- Implement and execute automated test cases, ensuring comprehensive coverage of business requirements and functionality.
- Integrate automated testing processes with QA tools such as Jira, TestRail, and Confluence for efficient test management and reporting.
- Analyze, troubleshoot, and resolve automation failures in collaboration with development and QA teams.
- Write and optimize SQL queries to validate data integrity and support automated test scenarios.
- Document test cases, results, and best practices, fostering knowledge sharing and process improvements within the team.
- Actively participate in Agile ceremonies, sprint planning, and defect triaging to ensure timely delivery of high-quality software.
- Continuously evaluate and recommend new tools, frameworks, and methodologies to advance automation capabilities.

## Skill Requirements
- Bachelor's degree in Computer Science, Information Technology, or a related field.
- Proven experience in Python automation, particularly with Selenium for web application testing.
- Proficiency in test management tools such as Jira, TestRail, and Confluence.
- Strong knowledge of SQL for data validation and analysis.
- Solid understanding of software testing methodologies, automation frameworks, and best practices.
- Excellent problem-solving, analytical, and communication skills.
- Ability to work collaboratively in a dynamic, fast-paced team environment.

## Other Requirements
- Industry certifications in automation testing (e.g., ISTQB, Certified Selenium Professional) are a plus.
- Experience with CI/CD pipelines and DevOps practices.
- Familiarity with cloud platforms and microservices architectures.
- Exposure to performance testing and security testing tools.
