## Job Title
Machine Learning Engineer

## Location
NOIDA

## Client Industry
Technology

## Detailed Responsibilities
- Design, develop, and deploy machine learning models and algorithms to solve complex business problems.
- Conduct thorough data analysis and preprocessing to ensure high-quality input for machine learning models.
- Collaborate with cross-functional teams to gather requirements, define project objectives, and deliver data-driven solutions.
- Implement various machine learning techniques, including supervised and unsupervised learning, regression, classification, clustering, and neural networks.
- Optimize and fine-tune models for performance, scalability, and robustness.
- Conduct model evaluation and validation using appropriate metrics and techniques.
- Develop and maintain machine learning pipelines and workflows for continuous integration and deployment.
- Utilize tools such as TensorFlow, PyTorch, Scikit-learn, and Pandas for model development and data manipulation.
- Stay up-to-date with the latest advancements in machine learning and artificial intelligence, and apply relevant techniques to projects.
- Provide technical guidance and mentorship to junior team members.

## Skill Requirements
- Strong proficiency in Python and experience with machine learning libraries such as TensorFlow, PyTorch, Scikit-learn, and Keras.
- Solid understanding of machine learning algorithms and principles, including supervised and unsupervised learning, regression, classification, clustering, and neural networks.
- Experience with data preprocessing, feature engineering, and feature selection techniques.
- Familiarity with data visualization tools such as Matplotlib, Seaborn, and Plotly.
- Experience with big data technologies such as Hadoop, Spark, and Apache Flink is a plus.
- Knowledge of cloud platforms such as AWS, Google Cloud, and Azure for deploying machine learning models.
- Strong problem-solving skills and the ability to think critically and analytically.
- Excellent communication and teamwork skills.
- Bachelor’s or Master’s degree in Computer Science, Data Science, Statistics, or a related field.

## Other Requirements
- Experience with natural language processing (NLP) and computer vision.
- Knowledge of deep learning frameworks and techniques.
- Experience with model interpretability and explainability techniques.
- Familiarity with version control tools such as Git.
- Strong background in mathematics and statistics.
