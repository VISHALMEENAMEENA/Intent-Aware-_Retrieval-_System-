## Detailed Responsibilities
- Develop and maintain user interfaces using React.js and Next.js frameworks.
- Implement responsive designs with Tailwind CSS to ensure applications are mobile-friendly and visually appealing.
- Collaborate with UX/UI designers to integrate user-centric design principles.
- Optimize components for maximum performance across a vast array of web-capable devices and browsers.
- Participate in code reviews and contribute to team knowledge-sharing sessions.
- Troubleshoot and resolve issues, ensuring the functionality and stability of applications.
- Stay updated with emerging trends and technologies in web development to bring innovative solutions to the team.

## Skill Requirements
- Bachelor’s degree in Computer Science, Information Technology, or related field.
- 2-5 years of professional experience in React.js and Next.js development.
- Strong proficiency in JavaScript, including DOM manipulation and the JavaScript object model.
- Experience with Tailwind CSS for crafting responsive and adaptive designs.
- Familiarity with RESTful APIs and asynchronous request handling.
- Knowledge of modern authorization mechanisms, such as JSON Web Token.
- Excellent problem-solving skills and attention to detail.

## Other Requirements

- Familiarity with version control systems like Git.
- Understanding of Agile methodologies and experience working in Agile teams.
