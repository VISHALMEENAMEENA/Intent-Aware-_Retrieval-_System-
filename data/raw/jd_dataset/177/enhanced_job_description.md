## Job Title
Senior .NET Developer – Microservices Specialist

## Location
N/A

## Client Industry
Information Technology & Services

## Detailed Responsibilities
- Design, develop, and deploy microservices using .NET Core to support enterprise-grade applications.
- Collaborate with product managers, architects, and front-end developers to define and implement software requirements.
- Integrate Object-Relational Mapping (ORM) tools for efficient data management and persistence.
- Ensure code quality through unit testing, code reviews, and adherence to best practices.
- Optimize application performance, scalability, and security within cloud or on-premise environments.
- Contribute to continuous integration and delivery (CI/CD) pipelines for seamless deployment.
- Document technical specifications, architectural decisions, and user guides.
- Mentor junior developers and provide technical guidance within the team.
- Stay abreast of emerging technologies and recommend improvements to existing systems.

## Skill Requirements
- Minimum 5 years of professional experience as a .NET Developer.
- Proven expertise in .NET Core and microservices architecture.
- Strong working knowledge of ORM tools (e.g., Entity Framework, Dapper).
- Proficient in C#, ASP.NET, and RESTful API development.
- Experience with source control systems such as Git.
- Familiarity with containerization tools (Docker, Kubernetes) is a plus.
- Bachelor’s degree in Computer Science, Engineering, or related field.
- Excellent problem-solving and analytical skills.
- Strong communication and teamwork abilities.

## Other Requirements
- Knowledge of cloud platforms (Azure, AWS, GCP) is an advantage.
- Experience with Agile/Scrum methodologies.
- Exposure to DevOps practices and CI/CD tools.
- Relevant certifications in .NET, cloud technologies, or microservices architecture.
