## Job Title
Senior React Developer (Next.js & Tailwind CSS) – eCommerce

## Location
N/A

## Client Industry
Information Technology / Multinational Technology Solutions Provider

## Detailed Responsibilities
- Design, develop, and optimize robust eCommerce web applications using React.js, Next.js, and Tailwind CSS.
- Collaborate with product managers, UX designers, and backend developers to deliver intuitive front-end solutions that meet business and user requirements.
- Write clean, maintainable, and efficient code while adhering to best practices in front-end development, performance optimization, and security.
- Implement responsive designs that provide a flawless user experience across desktop and mobile devices.
- Integrate APIs and third-party services to enrich application functionality and support business processes.
- Conduct code reviews, troubleshoot, and resolve technical issues, ensuring software quality and reliability.
- Stay updated with emerging front-end technologies and industry trends, proactively recommending enhancements and innovations.
- Participate in agile ceremonies, contribute to sprint planning, and support continuous integration and deployment workflows.
- Mentor and guide junior developers, fostering a culture of knowledge-sharing and technical excellence.

## Skill Requirements
- Bachelor’s or Master’s degree in Computer Science, Information Technology, or a related field.
- Minimum 5 years of hands-on experience in building large-scale eCommerce applications using React.js.
- Proven expertise with Next.js for server-side rendering and building scalable React applications.
- Strong proficiency with Tailwind CSS for utility-first, responsive UI development.
- Solid understanding of modern web technologies: HTML5, CSS3, JavaScript (ES6+), and RESTful APIs.
- Experience with version control systems (e.g., Git) and collaborative development workflows.
- Familiarity with agile methodologies and CI/CD pipelines.
- Excellent problem-solving skills, attention to detail, and effective communication abilities.

## Other Requirements
- Experience with headless CMS or eCommerce platforms (e.g., Shopify, Magento, CommerceTools) is a plus.
- Knowledge of TypeScript and testing frameworks (e.g., Jest, React Testing Library) is advantageous.
- Exposure to cloud environments (AWS, Azure, GCP) and DevOps practices.
- Previous experience working in a multinational, cross-cultural environment.
