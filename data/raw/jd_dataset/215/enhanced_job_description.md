## Detailed Responsibilities
- Design, develop, and maintain backend applications using Python, Flask, and Django frameworks.
- Develop RESTful APIs and integrate with third-party services and internal systems.
- Implement and optimize SQL queries and database schemas for high performance and scalability.
- Deploy, manage, and monitor applications on AWS cloud infrastructure.
- Build and maintain continuous integration/continuous deployment (CI/CD) pipelines using Jenkins and related tools.
- Utilize Jira for project tracking and GitHub for version control, ensuring seamless collaboration and code quality.
- Troubleshoot, debug, and resolve technical issues across the application lifecycle.
- Collaborate with frontend developers, DevOps engineers, and QA teams to deliver end-to-end solutions.
- Adhere to best practices in coding, testing, and documentation to ensure maintainability and scalability.
- Participate in code reviews, knowledge-sharing sessions, and contribute to process improvement initiatives.

- Proficiency in AWS services for application deployment and management.
- Experience with CI/CD tools, preferably Jenkins.
- Familiarity with project management and issue tracking tools such as Jira.
- Proficient in version control using GitHub.
- Strong problem-solving skills, attention to detail, and ability to work in a collaborative, fast-paced environment.
