## Detailed Responsibilities


## Skill Requirements
- Bachelor's degree in Computer Science, Information Technology, or related field (or equivalent work experience).
- Previous experience in a similar technical support role, preferably in a large enterprise environment.
- Strong knowledge of file server operating systems (such as Windows Server, Linux) and print server management.
- Familiarity with network protocols, file sharing technologies (e.g., SMB, NFS), and printer hardware.
- Excellent troubleshooting skills and the ability to analyze and resolve complex technical issues.
- Certifications such as Microsoft Certified: Windows Server, CompTIA Network+, or similar, are often preferred.
- Strong communication and interpersonal skills, with the ability to work well in a team and interact with end-users effectively.

## Other Requirements
- Experience with cloud-based storage solutions and advanced scripting abilities can be advantageous.
- Knowledge of configuration management tools and workflows is a plus.
