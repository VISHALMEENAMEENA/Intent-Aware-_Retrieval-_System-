* Develop, test, and deploy responsive web applications using React, Next.js, and Tailwind CSS, ensuring high-quality, scalable, and maintainable code.
* Lead or contribute to migration projects, specifically transitioning applications from Angular to React, managing migration challenges effectively.
* Collaborate closely with UX/UI designers, backend developers, and product owners to deliver seamless user experiences.
* Write clean, maintainable, and efficient code adhering to best practices and coding standards.
* Optimize application performance, including server-side rendering and static site generation, to ensure fast load times and responsiveness.
* Participate in code reviews, provide constructive feedback, and mentor junior developers as needed.
* Troubleshoot, debug, and resolve complex technical issues across multiple browsers and devices.
* Maintain clear documentation for code, processes, and architectural decisions.
* Stay up-to-date with the latest frontend trends, tools, and technologies to drive continuous improvement.
* Implement and adhere to best practices in version control, branching, and collaboration using Git.
* Integrate frontend components with backend services through RESTful APIs, ensuring robust and secure data exchange.
* Contribute to Agile development processes, participating actively in sprints, planning, and retrospectives.
* Apply analytical and problem-solving skills to troubleshoot and resolve technical issues efficiently.
* Foster effective communication and collaboration within cross-functional teams, conveying technical concepts clearly and professionally.
* Stay informed about emerging frontend technologies, frameworks, and methodologies to enhance development practices.

## Skill Requirements
* Bachelor’s degree in Computer Science, Information Technology, or a related field (or equivalent practical experience).
* Proven experience in developing web applications using React.js, Next.js, and Tailwind CSS, with a strong focus on component design and state management.
* Strong understanding of JavaScript (ES6+), HTML5, and CSS3, with proficiency in modern JavaScript features.
* Experience with version control systems (e.g., Git), including branching, merging, and collaborative workflows.
* Familiarity with RESTful API integration and consuming backend services within frontend applications.
* Demonstrated ability to work effectively within Agile development environments, participating in iterative development cycles.
* Excellent analytical, problem-solving, and debugging skills to troubleshoot complex technical issues.
* Effective communication and collaboration skills, capable of working with cross-functional teams and conveying technical information clearly.
* Experience in migrating applications from Angular to React, understanding migration challenges and solutions.
* Knowledge of TypeScript and ability to incorporate static typing into JavaScript codebases.
* Experience with state management libraries such as Zustand or Redux for scalable frontend architectures.
* Exposure to CI/CD pipelines and automated testing frameworks to ensure continuous integration and delivery.
* Prior experience working in multinational or large-scale enterprise environments, understanding enterprise development processes.
