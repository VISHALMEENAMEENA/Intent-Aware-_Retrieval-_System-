## Job Title
Azure Data Engineer

## Location
Spain

## Client Industry
Any Industry

## Detailed Responsibilities
- Design, develop, and optimize data pipelines leveraging Azure Data Factory to extract, transform, and load (ETL) data from diverse sources.
- Collaborate closely with data engineers, analysts, and stakeholders to analyze integration needs and optimize pipeline performance.
- Implement and enforce data security and compliance standards within the Azure Data Factory environment.
- Troubleshoot and resolve issues related to data pipelines and SQL databases to ensure seamless data operations.
- Create and maintain comprehensive technical documentation for all data pipelines and related processes.
- Stay abreast of the latest Azure Data Factory features, industry best practices, and emerging technologies to continually improve data integration workflows.
- Participate actively in Agile-Scrum teams, contributing to iterative development and continuous improvement initiatives.

## Skill Requirements
- Proven proficiency with Azure Data Factory (ADF) for the development and management of scalable data pipelines.
- Strong expertise in SQL, including data querying, manipulation, and performance optimization.
- Solid understanding of data modeling, ETL concepts, and database design principles.
- Experience working with Microsoft Azure cloud services and related data technologies.
- Excellent analytical, problem-solving, and troubleshooting abilities.
- Strong communication and interpersonal skills for effective collaboration in cross-functional teams.
- Bachelor’s degree in Computer Science, Information Technology, Engineering, or a related field (or equivalent work experience).

## Other Requirements
- Microsoft Certified: Azure Data Engineer Associate certification is preferred.
- Experience working in Agile-Scrum environments is advantageous.
- Familiarity with additional cloud data services and BI tools is a plus.
