## Job Title
Developer – JavaScript, React, CSS

## Location
Switzerland

## Client Industry
Technology Services (Applicable to Any Industry)

## Detailed Responsibilities
- Develop and implement new user-facing features using React.js, ensuring seamless integration and high-quality performance.
- Build reusable components and front-end libraries to streamline future development and maintain code consistency.
- Translate UI/UX designs and wireframes into responsive, well-structured code, maintaining best practices in web standards.
- Optimize application components for maximum speed, scalability, and compatibility across a range of devices and browsers.
- Collaborate with back-end developers and UX/UI designers to improve application usability and deliver cohesive solutions.
- Participate in code reviews, contribute to continuous improvement processes, and ensure adherence to project timelines and quality standards.
- Troubleshoot, debug, and upgrade existing web applications to enhance performance and user satisfaction.
- Maintain clear and comprehensive documentation for developed features and technical solutions.

## Skill Requirements
- Proficiency in JavaScript, React.js, and CSS, with a strong understanding of core front-end development principles.
- Hands-on experience with popular React.js workflows (such as Redux) and component-driven architecture.
- Familiarity with RESTful APIs and integration of front-end applications with back-end services.
- Knowledge of modern authorization mechanisms, such as JSON Web Token (JWT).
- Ability to interpret business requirements and translate them into effective technical solutions.
- Strong problem-solving skills, attention to detail, and the ability to work independently or as part of a team.
- Bachelor’s degree in Computer Science, Information Technology, or a related field (or equivalent work experience).

## Other Requirements
- React Developer Certification or similar credentials (preferred but not mandatory).
- Excellent written and verbal communication skills, with the ability to document technical solutions effectively.
- Experience working in agile development environments and collaborating with international teams.
