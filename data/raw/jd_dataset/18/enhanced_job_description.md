## Job Title
Senior Developer – Spring Boot, Python, Java

## Location
Noida, India

## Client Industry
Technology and IT Services

## Detailed Responsibilities
- Design, develop, and maintain enterprise-grade software applications using Spring Boot, Python, and Java.
- Collaborate with cross-functional teams—including Product Management, QA, and DevOps—to define and implement new features and enhancements.
- Participate actively in the software development lifecycle, including requirements analysis, coding, unit testing, integration, and deployment.
- Troubleshoot, debug, and optimize existing systems to ensure high performance and reliability.
- Conduct and participate in code reviews to uphold best practices and coding standards.
- Develop and maintain clear technical documentation for applications and processes.
- Implement RESTful web services and microservices architectures for scalable solutions.
- Integrate with relational and NoSQL databases such as MySQL, PostgreSQL, and MongoDB.
- Stay abreast of industry trends, emerging technologies, and best practices, proactively applying knowledge to improve processes and solutions.

## Skill Requirements
- Proven experience in developing applications using the Spring Boot framework.
- Strong proficiency in Python for backend development and data processing.
- In-depth expertise in Java and its ecosystem, including object-oriented programming and design patterns.
- Hands-on experience with RESTful APIs and microservices architecture.
- Familiarity with database technologies: MySQL, PostgreSQL, MongoDB.
- Good understanding of front-end technologies: HTML, CSS, JavaScript.
- Excellent problem-solving, analytical, and debugging skills.
- Ability to work independently and effectively within collaborative teams.
- Bachelor’s degree in Computer Science, Information Technology, or related field (or equivalent professional experience).
- Relevant certifications in Spring Boot, Python, or Java are advantageous.

## Other Requirements
- Experience with cloud platforms (AWS, Azure, GCP) and CI/CD pipelines is a plus.
- Exposure to Agile methodologies and DevOps practices.
- Strong communication and interpersonal skills to engage with global teams and clients.
