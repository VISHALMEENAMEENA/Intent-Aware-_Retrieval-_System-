## Job Title
Java J2EE Senior Tech Lead

## Location
Chennai or Noida

## Client Industry
Multinational Technology Provider

## Detailed Responsibilities
- Lead and mentor a team of developers, providing technical guidance and support to ensure successful project delivery.
- Design, develop, and deploy applications using Java J2EE, Microservices, and Spring Boot frameworks.
- Collaborate with cross-functional teams to gather requirements, define technical specifications, and deliver solutions that meet business needs.
- Implement best practices in application design, development, and deployment, ensuring high performance, scalability, and security.
- Participate in code reviews, identify areas for improvement, and implement solutions to enhance code quality.
- Utilize Agile/Scrum methodologies to manage project timelines, deliverables, and team collaboration.
- Troubleshoot and resolve technical issues, providing timely and effective solutions to ensure project continuity.
- Stay updated with the latest industry trends and emerging technologies to drive continuous improvement and innovation.

## Skill Requirements
- Bachelor's degree in Computer Science, Information Technology, or a related field.
- Minimum of 1+ years of experience in Java J2EE development.
- Strong leadership skills with the ability to manage and mentor a team.
- Proficiency in designing and developing applications using Java J2EE, Microservices, and Spring Boot.
- Experience with Agile/Scrum methodologies.
- Excellent problem-solving skills and the ability to think critically and creatively.
- Strong communication skills, both written and verbal, with the ability to convey technical concepts to non-technical stakeholders.

## Other Requirements
- Experience with cloud platforms such as AWS, Azure, or Google Cloud.
- Familiarity with containerization technologies like Docker and Kubernetes.
- Knowledge of DevOps practices and tools.
- Certifications in Java or related technologies.
