## Detailed Responsibilities
- Lead the creation of technical designs and solutions encompassing data, integrations, custom development, and DevOps.
- Develop advanced front-end/UI components using TypeScript, React JS, NextJS, or VueJS.
- Architect and implement robust back-end solutions using NodeJS, Python-Flask, Django, C# - .NET core, or Java Spring boot.
- Ensure the security of applications through Oauth API security, cryptography, tokens, JWTs, and vulnerability remediation.
- Act as a hands-on Tech Lead, engaging in coding, debugging, issue resolution, and production deployments.
